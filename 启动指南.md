# MES-Springboot 项目启动步骤

## 第一步：启动 MySQL 数据库

### 方法一：使用命令行启动

1. 按下 `Win + R`，输入 `powershell`，回车打开 PowerShell

2. 检查 MySQL 服务状态：
   ```powershell
   Get-Service *mysql*
   ```

3. 启动 MySQL 服务：
   ```powershell
   Start-Service MySQL80
   ```

### 方法二：使用服务管理器启动

1. 按下 `Win + R`，输入 `services.msc`，回车

2. 在服务列表中找到 `MySQL80`

3. 右键点击，选择「启动」

---

## 第二步：启动 MES 项目

### 1. 打开 PowerShell 终端

按下 `Win + R`，输入 `powershell`，回车

### 2. 切换到项目目录

```powershell
cd D:\HuaweiMoveData\Users\Zanna\Desktop\MES-Springboot\mes
```

### 3. 停止之前的 Java 进程（如有）

```powershell
Stop-Process -Name java -Force -ErrorAction SilentlyContinue
```

### 4. 启动项目

```powershell
C:\tools\apache-maven-3.8.8\bin\mvn.cmd clean spring-boot:run
```

### 5. 等待启动完成

当终端显示以下信息时，表示启动成功：

```
Tomcat started on port(s): 8080 (http)
Started SparchetypeApplication in X.X seconds
```

## 停止项目

在项目运行窗口按 `Ctrl + C` 停止项目

## 重新编译命令
mvn compile

## 终端进入数据库命令
mysql -u root -p

## 排错
强制杀掉所有 Java 进程：

taskkill /f /im java.exe

然后在 D:\HuaweiMoveData\Users\Zanna\Desktop\MES-Springboot\mes 目录重新启动，

mvn clean spring-boot:run

如果你不想 kill 所有 Java 进程（可能还有其他 Java 程序在跑），可以先找出占用端口的具体进程：

netstat -ano | findstr :8080

记下 PID，然后用 taskkill /f /pid <PID号> 杀掉它
