# MES系统功能报告

## 一、概述

章鱼师兄MES系统是一款基于Spring Boot + MyBatis-Plus + Redis + Shiro + Layui技术栈开发的制造执行系统。系统涵盖系统管理、工艺管理、基础数据管理、订单管理、计划管理和数字化展示等核心模块，为制造企业提供全面的生产管理解决方案。

---

## 一、常规管理

### 1.1 系统管理

#### 1.1.1 用户管理

**功能说明**：管理系统所有用户的基本信息，包括新增、编辑、删除、查询等操作。

**核心功能**：
- 用户列表分页查询，支持按姓名、用户名模糊搜索
- 用户新增与编辑，包含姓名、用户名、部门、邮箱、手机号、性别、出生日期等字段
- 用户删除（支持单个删除和批量删除），同时清理用户角色关联
- 当前登录用户信息查询
- 用户基本资料修改
- 用户密码修改（需验证原密码）
- 用户头像上传

**技术实现**：
- 密码采用MD5哈希加密，盐值为用户名，加密3次
- 头像存储于本地文件系统
- 使用Shiro进行身份认证

**接口地址**：`/admin/sys/user/list-ui`

**相关文件**：
- 控制器：[SysUserController.java](file:///d:/Desktop/MES-Springboot/mes/src/main/java/com/wangziyang/mes/system/controller/admin/SysUserController.java)
- 实体类：[SysUser.java](file:///d:/Desktop/MES-Springboot/mes/src/main/java/com/wangziyang/mes/system/entity/SysUser.java)
- 服务类：ISysUserService

**功能说明**：管理系统角色及其权限分配，实现基于角色的访问控制（RBAC）。

- 角色权限分配（菜单权限、数据范围权限）

**技术实现**：
- 角色与菜单通过中间表 SysRoleMenu 关联
- 角色与用户通过中间表 SysUserRole 关联
- 支持按机构进行数据范围权限划分

#### 1.1.3 菜单管理

**功能说明**：配置系统菜单结构、操作权限和按钮权限标识。
