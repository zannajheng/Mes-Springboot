# MES系统操作指导书

## 概述

本指导书旨在帮助用户快速掌握章鱼师兄MES系统的各项功能操作。系统涵盖常规管理、工艺管理、基础数据管理、订单管理、计划管理和数字化展示等核心模块。本文档将逐一介绍每个功能的操作步骤，便于用户上手使用。

---

## 一、常规管理

### 1.1 系统管理

#### 1.1.1 用户管理

**功能入口**：左侧导航栏 → 常规管理 → 系统管理 → 用户管理

**操作步骤**：

1. **查询用户**
   - 在列表页面顶部的"姓名"或"用户名"输入框中输入查询条件
   - 点击"查询"按钮，系统将按条件筛选并展示用户列表
   - 点击"重置"按钮可清空查询条件并刷新列表

2. **新增用户**
   - 点击列表上方的"新增"按钮
   - 在弹出的表单中填写：姓名、用户名、密码、部门、邮箱、手机号、性别、出生日期
   - 勾选需要分配的角色（可多选）
   - 点击"确定"保存用户信息

3. **编辑用户**
   - 在用户列表中找到目标用户，点击对应行的"编辑"按钮
   - 修改需要变更的字段信息
   - 点击"确定"保存修改

4. **删除用户**
   - 单个删除：点击目标用户行的"删除"按钮，在确认弹窗中点击"确定"
   - 批量删除：勾选多个用户前的复选框，点击列表上方"批量删除"按钮，确认后删除

5. **修改个人资料**
   - 点击页面右上角用户头像，选择"基本资料"
   - 修改姓名、邮箱、手机号、性别等信息
   - 点击"保存"完成修改

6. **修改密码**
   - 点击页面右上角用户头像，选择"修改密码"
   - 输入原密码、新密码、确认新密码
   - 点击"保存"完成密码修改

7. **上传头像**
   - 在个人资料页面点击头像
   - 选择本地图片文件（支持jpg、png格式）
   - 确认上传后头像自动更新

#### 1.1.2 角色管理

**功能入口**：左侧导航栏 → 常规管理 → 系统管理 → 角色管理

**操作步骤**：

1. **查询角色**
   - 在列表顶部输入角色名称或角色编码
   - 点击"查询"按钮查看匹配的角色列表

2. **新增角色**
   - 点击"新增"按钮
   - 填写角色名称、角色编码、备注信息
   - 点击"确定"保存

3. **编辑角色**
   - 点击目标角色行的"编辑"按钮
   - 修改角色信息后点击"确定"保存

4. **删除角色**
   - 点击目标角色行的"删除"按钮
   - 确认后系统将删除该角色及关联数据

5. **分配权限**
   - 点击目标角色行的"权限分配"按钮
   - 在弹出的权限树中勾选该角色可访问的菜单
   - 选择数据范围（全部、本部门、本部门及下级、仅本人）
   - 点击"保存"完成权限分配

#### 1.1.3 菜单管理

**功能入口**：左侧导航栏 → 常规管理 → 系统管理 → 菜单管理

**操作步骤**：

1. **查询菜单**
   - 在列表顶部输入菜单名称
   - 点击"查询"按钮查看匹配的菜单列表

2. **新增菜单**
   - 点击"新增"按钮
   - 选择菜单类型：目录、菜单、按钮
   - 填写菜单名称、父菜单（可选）、路由地址、权限标识
   - 设置排序号（数字越小排序越靠前）
   - 点击"确定"保存

3. **编辑菜单**
   - 点击目标菜单行的"编辑"按钮
   - 修改菜单信息后点击"确定"保存

4. **删除菜单**
   - 点击目标菜单行的"删除"按钮
   - 确认后删除该菜单（若有子菜单需先删除子菜单）

#### 1.1.4 部门管理

**功能入口**：左侧导航栏 → 常规管理 → 系统管理 → 部门管理

**操作步骤**：

1. **查询部门**
   - 在部门树中选择目标部门，右侧列表展示该部门及下级部门用户
   - 可按部门名称进行搜索

2. **新增部门**
   - 点击"新增部门"按钮
   - 填写部门名称、父部门、排序号
   - 点击"确定"保存

3. **编辑部门**
   - 点击目标部门的"编辑"按钮
   - 修改部门信息后点击"确定"保存

4. **删除部门**
   - 点击目标部门的"删除"按钮
   - 确认后删除该部门（需先清空部门下用户）

#### 1.1.5 字典管理

**功能入口**：左侧导航栏 → 常规管理 → 系统管理 → 字典管理

**操作步骤**：

1. **查询字典**
   - 在列表顶部输入字典名称或字典编码
   - 点击"查询"按钮查看匹配字典

2. **新增字典**
   - 点击"新增"按钮
   - 填写字典名称、字典编码、备注
   - 点击"确定"保存

3. **字典项管理**
   - 点击目标字典行的"字典项"按钮
   - 在字典项列表中可新增、编辑、删除字典项
   - 填写字典项标签、字典项值、排序号
   - 点击"保存"完成操作

#### 1.1.6 权限管理

**功能入口**：左侧导航栏 → 常规管理 → 系统管理 → 权限管理

**操作步骤**：

1. **查询权限**
   - 在列表顶部输入权限名称
   - 点击"查询"按钮查看匹配权限

2. **新增权限**
   - 点击"新增"按钮
   - 填写权限名称、父权限、权限标识
   - 点击"确定"保存

3. **编辑权限**
   - 点击目标权限行的"编辑"按钮
   - 修改权限信息后点击"确定"保存

4. **删除权限**
   - 点击目标权限行的"删除"按钮
   - 确认后删除该权限

### 1.2 异常页面

#### 1.2.1 404页面

**功能入口**：访问不存在的URL时自动跳转

**操作步骤**：
- 当用户访问系统中不存在的页面时，系统自动跳转到404错误页面
- 点击"返回首页"按钮可回到系统主页面

### 1.3 Swagger

#### 1.3.1 API文档

**功能入口**：左侧导航栏 → 常规管理 → Swagger → API文档

**操作步骤**：
- 在API文档页面左侧可展开各个模块的接口列表
- 点击具体接口可查看接口详细说明、请求参数、响应格式
- 点击"Try it out"按钮可在线测试接口
- 填写必要参数后点击"Execute"执行测试
- 在下方可查看接口返回结果

## 二、组件管理

### 2.1 图标列表

**功能入口**：左侧导航栏 → 组件管理 → 图标列表

**操作步骤**：
- 进入页面后系统展示所有可用图标
- 可通过滚动浏览所有图标
- 点击目标图标可复制图标类名到剪贴板

### 2.2 图标选择

**功能入口**：左侧导航栏 → 组件管理 → 图标选择

**操作步骤**：
- 在搜索框中输入图标名称进行搜索
- 从搜索结果中选择需要的图标
- 选中后系统自动填充图标类名

### 2.3 颜色选择

**功能入口**：左侧导航栏 → 组件管理 → 颜色选择

**操作步骤**：
- 从预设颜色面板中选择需要的颜色
- 也可在颜色输入框中输入自定义颜色值
- 点击"确定"完成颜色选择

### 2.4 分步表单

**功能入口**：左侧导航栏 → 组件管理 → 分步表单

**操作步骤**：
- 按照步骤提示依次填写每个步骤的表单内容
- 填写完当前步骤后点击"下一步"进入下一阶段
- 在任意步骤可点击"上一步"返回修改
- 全部步骤完成后点击"提交"保存表单数据

### 2.5 富文本编辑器

**功能入口**：左侧导航栏 → 组件管理 → 富文本编辑器

**操作步骤**：
- 在编辑器中输入或粘贴文本内容
- 使用工具栏按钮进行格式化操作：加粗、斜体、下划线、字号调整、颜色设置
- 可插入图片、表格、链接等元素
- 点击"获取内容"按钮可获取编辑器中的HTML内容

### 2.6 登录页面示例

**功能入口**：左侧导航栏 → 组件管理 → 登录页面示例

**操作步骤**：
- 登录-1：经典风格登录页面，填写用户名、密码、验证码后点击"登录"
- 登录-2：现代风格登录页面，支持账号密码登录和扫码登录切换

## 三、工艺管理

### 3.1 工艺路线管理

**功能入口**：左侧导航栏 → 工艺管理 → 工艺路线管理

**操作步骤**：

1. **查询工艺路线**
   - 在列表顶部输入路线名称或路线编号
   - 点击"查询"按钮查看匹配的工艺路线

2. **新增工艺路线**
   - 点击"新增"按钮
   - 填写工艺路线名称、描述信息
   - 在工序列表中添加工序，填写工序编号、工序名称、加工内容
   - 设置工序顺序，点击"确定"保存

3. **编辑工艺路线**
   - 点击目标路线行的"编辑"按钮
   - 修改路线信息或工序信息
   - 点击"确定"保存

4. **删除工艺路线**
   - 点击目标路线行的"删除"按钮
   - 确认后删除该工艺路线

### 3.2 工艺BOM管理

**功能入口**：左侧导航栏 → 工艺管理 → 工艺BOM管理

**操作步骤**：

1. **查询BOM**
   - 在列表顶部输入物料编码或BOM名称
   - 点击"查询"按钮查看匹配的BOM列表

2. **新增BOM**
   - 点击"新增"按钮
   - 填写BOM名称、版本号、物料编码
   - 在BOM子项列表中添加子物料，填写子物料编码、数量、单位
   - 关联工艺路线
   - 点击"确定"保存

3. **编辑BOM**
   - 点击目标BOM行的"编辑"按钮
   - 修改BOM信息或子项信息
   - 点击"确定"保存

4. **删除BOM**
   - 点击目标BOM行的"删除"按钮
   - 确认后删除该BOM及其子项

5. **生成物料编码**
   - 在新增BOM时点击"生成编码"按钮
   - 系统自动生成物料编码

### 3.3 工艺内容编制

**功能入口**：左侧导航栏 → 工艺管理 → 工艺内容编制

**操作步骤**：

1. **查询工艺内容**
   - 选择BOM编号进行筛选
   - 点击"查询"按钮查看对应的工艺内容列表

2. **初始化工艺内容**
   - 点击"初始化"按钮
   - 系统从BOM工艺关系继承数据，生成初始工艺内容

3. **编辑工艺内容**
   - 点击目标行的"编辑"按钮
   - 修改工序名称、工序内容、技术要求
   - 点击"保存"完成编辑

4. **完成编制**
   - 工艺内容编辑完成后点击"完成编制"按钮
   - 系统锁定工艺内容，不再允许修改

5. **工装设备管理**
   - 在工艺内容详情页切换到"工装设备"标签
   - 点击"添加工装"按钮，填写设备名称、规格、数量
   - 选择已有设备或新增设备后保存

6. **技术文档管理**
   - 在工艺内容详情页切换到"技术文档"标签
   - 点击"上传文档"按钮，选择本地文件上传
   - 填写文档名称、文档类型、备注信息
   - 点击"上传"完成

7. **备料清单管理**
   - 在工艺内容详情页切换到"备料清单"标签
   - 点击"添加备料"按钮，填写物料编码、数量、用途
   - 点击"保存"完成添加

8. **导出备料清单**
   - 在备料清单列表页面点击"导出"按钮
   - 系统将备料清单导出为Excel文件
   - 选择保存路径完成导出

### 3.4 零部件定义

**功能入口**：左侧导航栏 → 工艺管理 → 零部件定义

**操作步骤**：

1. **查询零部件**
   - 在列表顶部输入零件编号或零件名称
   - 点击"查询"按钮查看匹配的零部件列表

2. **新增零部件**
   - 点击"新增"按钮
   - 填写零件编号、零件名称、规格型号、材料
   - 填写加工要求、备注信息
   - 点击"确定"保存

3. **编辑零部件**
   - 点击目标零部件行的"编辑"按钮
   - 修改零件信息后点击"确定"保存

4. **删除零部件**
   - 点击目标零部件行的"删除"按钮
   - 确认后删除该零部件

5. **生成零件编号**
   - 在新增零部件时点击"生成编号"按钮
   - 系统自动生成零件编号

### 3.5 工序管理

**功能入口**：左侧导航栏 → 工艺管理 → 工序管理

**操作步骤**：

1. **查询工序**
   - 在列表顶部输入工序编号或工序描述
   - 点击"查询"按钮查看匹配的工序列表

2. **新增工序**
   - 点击"新增"按钮
   - 填写工序编号、工序名称、工序描述
   - 设置工序类型、默认工时
   - 点击"确定"保存

3. **编辑工序**
   - 点击目标工序行的"编辑"按钮
   - 修改工序信息后点击"确定"保存

4. **删除工序**
   - 点击目标工序行的"删除"按钮
   - 确认后删除该工序

### 3.6 工艺查询

**功能入口**：左侧导航栏 → 工艺管理 → 工艺查询

**操作步骤**：
- 在查询页面选择产品型号或BOM编号
- 点击"查询"按钮
- 系统展示该产品的完整工艺路线、工序详情、备料清单
- 可点击工序查看详细的工艺内容

### 常见问题

**问题1：忘记密码怎么办？**
答：联系系统管理员，在用户管理中重置密码。

**问题2：页面加载缓慢怎么办？**
答：检查网络连接，清理浏览器缓存，或联系系统运维人员。

**问题3：数据无法保存怎么办？**
答：检查必填项是否填写完整，网络是否正常，若问题持续请联系运维人员。

**问题4：如何申请新功能权限？**
答：联系系统管理员，在角色管理中分配相应权限。
